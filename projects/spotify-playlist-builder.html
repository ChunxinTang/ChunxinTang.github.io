Overview

Built with Python (Flask), EasyOCR, and the Spotify Web API, this project lets users upload a photo or screenshot of handwritten or printed song lists ‚Äî and automatically generates a Spotify playlist based on OCR-extracted text.

App Interface

Mobile App Interface

React Native mobile app for uploading and processing song lists

What it does

  • Accepts images via a web/mobile interface
  • Uses EasyOCR to detect and extract song titles and artists
  • Matches each song to Spotify using the API
  • Creates a public Spotify playlist and returns the link
